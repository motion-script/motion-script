# Directional Blur

Smears the node's content along a single axis — a linear, motion-blur-style streak. Unlike [Motion Blur](./motion-blur), the direction and length are authored manually rather than derived from the node's velocity.

![Directional blur effect demo](/effects/directional-blur.png)

## Usage

```tsx
import { Effects } from '@motion-script/core';

// Horizontal smear, 40px
<Rect effects={Effects.directionalBlur(0, 40)} />

// Diagonal smear
<Rect effects={Effects.directionalBlur(45, 20)} />

// Raw object
<Rect effects={[{ type: 'directionalBlur', direction: 0, blurLength: 40 }]} />
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `type` | `'directionalBlur'` | — | Effect identifier |
| `direction` | `number` | — | Smear angle in degrees (`0` = horizontal, `90` = vertical) |
| `blurLength` | `number` | — | Smear length in pixels |
| `backdrop` | `boolean` | `false` | When `true`, smears the content beneath the node instead of the node itself |

## Animating

Animate `blurLength` to create a speed-up or slow-down effect:

```tsx
import { Scene, Rect, Effects, tween, lerpNumber, easeIn } from '@motion-script/core';

export class MyScene extends Scene {
  *build() {
    const card = new Rect({ width: 500, height: 300, fill: 'royalblue' });
    card.set({ effects: Effects.directionalBlur(0, 0) });
    this.add(card);

    // Blur out to the right
    yield* tween(0.6, (t) => {
      card.set({ effects: Effects.directionalBlur(0, lerpNumber(0, 60, easeIn(t))) });
    });
  }
}
```

## Stacking with other effects

```tsx
// Motion-look: directional blur + chromatic aberration
<Rect effects={Effects.directionalBlur(0, 30).chromaticAberration(6, 0)} />
```
