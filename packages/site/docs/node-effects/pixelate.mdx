# Pixelate

Chunks the node into rectangular blocks.

## Usage

```tsx
import { FX } from '@motion-script/core';

// Uniform pixelation on both axes — block size in pixels
<Rect fill={{ type: 'image', src: './photo.jpg', mode: 'fill' }} effects={FX.pixelate(16)} />

// Raw object with separate axes
<Rect effects={[{ type: 'pixelate', horizontalBlocks: 24, verticalBlocks: 8 }]} />
```

## Props

| Prop | Type | Description |
|---|---|---|
| `type` | `'pixelate'` | Effect identifier |
| `horizontalBlocks` | `number` | Horizontal block size in pixels: 1 = no pixelation |
| `verticalBlocks` | `number` | Vertical block size in pixels: 1 = no pixelation |

The `FX.pixelate(size)` helper sets both axes to the same value.

## Animating — pixel reveal

Start highly pixelated and animate down to 1 to reveal the image:

```tsx
import { Scene, Rect, FX, tween, lerpNumber, easeInOut } from '@motion-script/core';

export class MyScene extends Scene {
  *build() {
    const img = new Rect({
      width: 600,
      height: 400,
      fill: { type: 'image', src: './photo.jpg', mode: 'fill' },
    });
    img.set({ effects: FX.pixelate(40) });
    this.add(img);

    yield* tween(1.5, (t) => {
      img.set({ effects: FX.pixelate(lerpNumber(40, 1, easeInOut(t))) });
    });
  }
}
```

## Stacking

```tsx
<Rect effects={FX.pixelate(12).grayscale(0.5)} />
```
