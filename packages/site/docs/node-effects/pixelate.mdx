# Pixelate

Chunks the node into rectangular blocks, like After Effects' **Mosaic** effect.

The block values are a **block count** — how many blocks span the node — *not* a
pixel size. A count equal to the node's pixel dimension on that axis is pristine
(one block per pixel); lower counts give larger, coarser blocks. For a 1920×1080
image, `1920 × 1080` is untouched and `200 × 180` is heavily pixelated.

## Usage

```tsx
import { FX } from '@motion-script/core';

// Uniform block count on both axes
<Rect fill={{ type: 'image', src: './photo.jpg', mode: 'fill' }} effects={FX.pixelate(120)} />

// Per-axis block counts + sharp colours
<Rect effects={FX.pixelate({ horizontalBlocks: 200, verticalBlocks: 180, sharpColors: true })} />

// Uniform count via an options object
<Rect effects={FX.pixelate({ blocks: 80, sharpColors: false })} />

// Raw effect object
<Rect effects={[{ type: 'pixelate', horizontalBlocks: 200, verticalBlocks: 180, sharpColors: true }]} />
```

`FX.pixelate(options)` accepts:

- `number` — same block count on both axes (`sharpColors` defaults to `true`).
- `{ blocks, sharpColors? }` — uniform block count.
- `{ horizontalBlocks, verticalBlocks, sharpColors? }` — per-axis block counts.

## Props

| Prop | Type | Description |
|---|---|---|
| `type` | `'pixelate'` | Effect identifier |
| `horizontalBlocks` | `number` | Number of blocks across the node (AE "Horizontal Blocks") |
| `verticalBlocks` | `number` | Number of blocks down the node (AE "Vertical Blocks") |
| `sharpColors` | `boolean` | AE "Sharp Colors": solid blocks (`true`) vs. smoothly blended (`false`) |

## Animating — pixel reveal

Start with a low block count (coarse) and animate up to the node's resolution to
resolve the image:

```tsx
import { Scene, Rect, FX, tween, lerpNumber, easeInOut } from '@motion-script/core';

export class MyScene extends Scene {
  *build() {
    const img = new Rect({
      width: 600,
      height: 400,
      fill: { type: 'image', src: './photo.jpg', mode: 'fill' },
    });
    img.set({ effects: FX.pixelate(40) });
    this.add(img);

    yield* tween(1.5, (t) => {
      img.set({ effects: FX.pixelate(lerpNumber(40, 600, easeInOut(t))) });
    });
  }
}
```

## Stacking

```tsx
<Rect effects={FX.pixelate(120).grayscale(0.5)} />
```
